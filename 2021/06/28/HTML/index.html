一. Web概念概述

  1. JavaWeb：使用Java语言开发基于互联网的项目
  2. 软件架构：
    1. C/S：Client/Server 客户端/服务器
      • 在用户本地有一个客户端程序，在远程有一个服务器程序
      • eg. QQ，迅雷…
      • 优点：用户体验好
      • 缺点：开发，安装，部署，维护麻烦
    2. B/S：Browser/Server 浏览器/服务器
      • 只需要一个浏览器，用户通过不同的网址(URL)，可以访问远程不同的服务器程序
      • 优点：开发，安装，部署，维护比较简单
      • 缺点：如果应用过大，用户体验可能会受到影响；对硬件要求过高

二. B/S架构详解

  1. 资源分类
    1. 静态资源：
      1. 概念：使用静态网页开发技术发布的资源
      2. 特点：所有用户访问，得到的结果都是一样的，比如图片，文本，音频，视频，HTML，CSS，Javascript
        • HTML：用于搭建基础网页，展示页面内容
        • CSS：用于美化页面，布局页面
        • Javascript：控制页面的元素，让页面有一些动态效果
      3. 如果用户请求的是静态资源，那么服务器会直接将静态资源发送给浏览器。浏览器内置的静态资源解析引擎，可以展示静态资源
    2. 动态资源：
      1. 概念：使用动态网页开发技术发布的资源
      2. 特点：所有用户访问，得到的结果可能不一样，比如 jsp/servlet，php，asp…
      3. 如果用户请求的是动态资源，那么服务器会执行动态资源，转换为静态资源之后，再发送给浏览器解析

三. HTML

  1. 概念：Hyper Text Markup Language 超文本标记语言，最基础的网页开发语言
    1. 超文本：超文本是用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本
    2. 标记语言：有标签构成的语言。<标签名称> 如html，xml；标记语言不是编程语言
  2. 快速入门：
    1. 语法
      1. html文档后缀名 .html 或者 .htm，两者无差别
      2. 标签分类
        1. 围堵标签：有开始标签和结束标签。如<html> </html>
        2. 自闭和标签：开始标签和结束标签在一起。如换行<br/>
      3. 标签可以嵌套
      4. 在开始标签中可以定义属性，属性是由键值对构成，值需要用引号(单双都可)引起来
      5. html的标签不区分大小写，建议都使用小写
  3. 标签：
    1. 文件标签：构成html最基本的标签
      1. html：html文档的根标签
      2. head：头标签，用于指定html文档的一些属性，引入外部的资源
      3. title：定义标题标签
      4. body：体标签
      5. <!DOCTYPE html>：html5种定义该文档是html文档
    2. 文本标签：和文本有关的标签
      1. <!-- 注释内容 -->
      2. <h1> to <h6>：自动换行 级数增大字体减小
      3. <p>：段落标签
      4. <br>：换行标签写<br>或者<br/>都可以
      5. <hr>：水平线 自闭和标签
        1. 属性：
          1. color
          2. width：宽度
          3. size：高度
          4. align：对齐方式 center left ritght
      6. <b>：字体加粗
      7. <i>：斜体
      8. <font>：字体标签
        1. 属性
          1. color：颜色
          2. size：大小
          3. face：字体
      9. <center>：文本居中标签
      10. 属性定义：
        1. color
          1. 英文单词
          2. rgb(值1，值2，值3)：值的范围：0~255
          3. #值1值2值3：值的范围：00~FF之间
        2. width
          1. 数值，单位是px像素
          2. 数值%：相对于父元素的比列宽度
    3. 图片标签：自闭和标签<img>
      1. 属性
        1. src：路径多数为相对路径 ；以 .开头的路径，./代表当前html文档所在路径；../代表当前html文档所在文件夹的上一级文件夹目录；图片文件夹images文件夹放在html文档所在的同一目录文件夹内的话，该html文档中的图片标签就可以写为<img src = "./images/XXX.jpg">或者<img src = "images/XXX.jpg">
        2. align：对齐方式
        3. alt：图片文字信息
        4. width
        5. height
    4. 列表标签，列表标签内嵌套<li>包裹列表中的每一项
      1. 有序列表<ol>的属性
        1. type：索引的种类
        2. start：索引的起始值
      2. 无序列表<ul>的属性
        1. type：符号类型
    5. 连接标签：<a>的属性
      1. href：表示超链接点击之后跳转的位置的URL(统一资源定位符)，也可以为本地资源链接；也可以发邮件，属性值为“mailto：邮箱地址”
      2. target属性值：
        1. 为_self：默认值，在本窗口页面内打开超链接
        2. 为_blank：在新的窗口打开超链接
      3. 超链接的功能：
        1. 显示可以被点击的样式：不写href属性，不显示此样式
        2. 点击之后跳转到href指定的url，若想点击之后不跳转可以使 href=”javascript:void(0);“
    6. <div>和<span>标签，div默认换行，span包裹的文本信息在一行展示，结合CSS来控制样式
    7. 语义化标签：<header>,<footer>等，h5中为了提高程序可读性，提供的一些标签
    8. 表格标签，先定义表格，再定义行，然后定义每行中有几个单元格
      1. <table>：定义表格
        1. border：规定表格是否拥有边框，值1
        2. width：宽度
        3. cellpadding：规定单元边沿与其内容之间的空白，值pixels
        4. cellspacing：规定单元格之间的空白，值pixels
        5. bgcolor：背景色
        6. align
      2. <tr>：定义行
        1. bgcolor：背景色
        2. align
      3. <td>：定义单元格
        1. bgcolor：背景色
        2. align
        3. colspan：合并列，值为1表示占1列，以此类推
        4. rowspan：合并行，如上
      4. <th>：定义表头单元格
      5. <caption>：定义表格标题，默认居中
      6. <thead>：表示表格的头部份，语义化标签
      7. <tbody>：表示表格的体部分
      8. <tfoot>：表示表格的脚部分
    9. 表单标签<form>：用于采集用户输入的数据，传递给服务器进行交互；用于定义一个表单，可以定义一个范围，范围代表采集用户数据的范围；表单项中的数据想要被提交，必须指定其name属性
      1. action：指定数据提交的URL
      2. method：指定提交方式；
        1. 值1get：请求参数会在地址栏显示，封装在请求行中；请求参数的长度有限制；不太安全。
        2. 值2post：请求参数不会再地址栏显示，会封装在请求体中；请求参数的长度无限制；较为安全。
    10. 表单项标签
      1. <input>：可以通过type属性值，改变元素的显示样式

        1. type属性的取值：

          1. text：文本输入框，默认为此值
          2. password：密码输入框
          3. radio：单选框
            • 注意：
              1. 要想让多个单选框实现单选的效果，则多个单选框的name属性值必须一样
              2. 一般会给每一个单选框提供value属性值，指定其被选中后提交的值
              3. checked属性，可以指定默认值
          4. chechbox：复选框
            • 注意：
              1. 一般会给每一个单选框提供value属性值，指定其被选中后提交的值
              2. checked属性，可以指定默认值
          5. file：文件选择框
          6. hidden：隐藏域，用value属性指定一些被提交的信息，但是网页上不会显示该信息
          7. submit：提交按钮
          8. button：普通按钮配合js使用
          9. image：图片按钮，可以提交表单 <input type = "image" src = "路径">
          10. color：取色器
          11. date，datetime-loacl
          12. email：自行正则校验
          13. number：只能输入数字

        2. value：指定的值在文本输入框中正常显示，需要自己手动删除

        3. placeholder：指定输入框的提示信息，当输入框的内容发生变化，会自动清空提示信息

      2. <lable>：指定输入项的文字描述信息，<lable>的for属性值一般会和<input>的id属性值对应，点击<lable>包裹的文本区域，会让同id属性值的<input>自动获取光标
      3. <select>：下拉列表
        1. <option>：包裹列表的选择项，添加selected则该条记录被默认选择，一般默认第一条
      4. <textarea>：文本域
        1. cols：列数，代表每一行有多少个字符
        2. rows：行数

四. CSS（页面美化和布局控制）

  1. 概念：Cascading Style Sheets，层叠样式表
    • 层叠：多个样式可以作用在同一个html的元素上，同时生效
  2. 好处：
    • 功能比较强大
    • 将内容展示和样式控制分离。降低耦合度，解耦；让分工更容易；提高开发效率
  3. CSS的使用：CSS与html结合的方式：
    • 内联样式：<div style = "color:red">CSS</div>，在标签内使用style属性指定css代码
    • 内部样式：在head标签内，定义style标签，style标签的内容就是css代码
    • 外部样式：
      1. 定义css资源文件
      2. 在head标签内，定义link标签，引入外部的资源文件

  1. CSS语法：

    1. 格式：

      选择器 {

      ​ 属性名1：属性值1；

      ​ 属性名2：属性值2；

      ​ …….

      }

      选择器：筛选具有相似特征的元素

  2. 选择器：筛选具有相似特征的元素

    • 分类

      1. 基础选择器

        1. id选择器：选择具体的id属性值元素，建议在一个html页面中id值唯一；

          语法：#id属性值{}

        2. 元素选择器：选择具有相同标签名称的元素；

          语法：标签名称{}

        3. 类选择器：选择具有相同的class属性值的元素,可以作用在多个标枪上

          语法：.class属性值{}

        4. 优先级：1 > 3 >2

      1. 扩展选择器:
        1. 选择所有元素：*{}
        2. 并集选择器：选择器1，选择器2{}；eg: div,p{color:red;}
        3. 子选择器：选择器1 选择器2{}；筛选选择器1下的选择器2
        4. 父选择器：选择器1> 选择器2{}; 筛选选择2的父元素选择器1
        5. 属性选择器：元素名称[属性名=”属性值“]{}；选择对应元素名称且属性名等于规定属性值的元素；eg：input[type = "text"]{ border:5px solid; }
        6. 伪类选择器：元素:状态{}；选择一些元素具有的状态
          1. eg：<a>的状态
            • link：初始化状态
            • visited：被访问过的状态，点击完松手后的状态
            • active：正在访问的状态，点击没松手的状态
            • hover：鼠标悬浮在上方的状态

  3. 属性：

    1. 字体，文本
      1. font-size：字体大小
      2. color：文本颜色
      3. text-align：文本对齐方式
      4. line-height：行高
    2. 背景
      1. background：background-image:url("图片路径") centenr no-repeat
    3. 边框
      1. border：设置边框，复合属性；border:1px solid red;
    4. 尺寸
      1. width
      2. height
    5. 盒子模型：控制布局
      1. margin：外边距
      2. padding：内边距
        • 默认情况下内边距会影响整个盒子的大小
        • box-sizing:border; 设置盒子的属性，让width和height就是最终盒子的大小
      3. float：浮动
        • left
        • right
